Sobre

Este é um site projetado para compartilhar reflexões trazidas durante e pós aula da disciplina Estrutura e Dinâmica Social em 2025.
Ele foi criado totalmente com base HTML e CSS sem uso de nenhuma linguagem ou framework extra, seu código pode ser encontrado em Github e foi hospedado por meio do site Netlify.
Os textos que se encontram nesse site são transcrições com alguns aprimoramentos daqueles escritos em sala de aula.
